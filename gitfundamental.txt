O QUE É GITHUB?
Github é um serviço para gerenciar reposítorios, gratuito e amplamente utilizado.
Pode ser enviado projetos para o GitHub e disponibilizá-lo para outros devs.
READ ME: É o arquivo que vai descrever o seu repositório.
GIT IGNORE: Arquivo para você escolher quais arquivos o git não deve monitorar.
LICENSE: Arquivo para você inserir uma licença ao seu repositório.
git init: para iniciar o seu repositório
git status: para verificar o status do seu repositório
git add: para adicionar aquivos ao projeto, pode adcionar um arquivo específico ou diversos de uma unica vez
git commit -m: para enviar o seu arquivo para o Github
git push -u origin master: para enviar o seu repositório.
git pull: recebe alterações feitos no repositório remoto para a sua máquina
git clone: usado para clonar o repositório que está no github e trazer para a sua máquina.
git rm: remove um arquivo do seu repositorio através do comando git rm <filename>
git log: Comando utilizado para verificar alterações no repositório
git mv: usado para mudar o nome do arquivo ou para mover o arquivo para outro local
    ex: git mv GitFundamental.txt gitfundamental.txt
git checkout: usado para desfazer alterações em um arquivo
    ex: git checkout gitfundamental.txt
git ignore: usado para fazer com que o git ignore um arquivo do projeto
    OBS: para usar, você cria um arquivo .gitignore e adiciona o nome do arquivo que deseja ignorar no seu projeto.
git reset: Comando utilizado para resetar todas as mudanças feitas. Geralmente usado com a flag --hard
